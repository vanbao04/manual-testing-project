# Manual Testing Project – Intern IT

## Giới thiệu
Đây là project manual testing cơ bản dành cho vị trí Intern IT / Intern Tester.
Mục tiêu của project là thực hành quy trình kiểm thử phần mềm và cách ghi nhận lỗi.

## Website được test
https://the-internet.herokuapp.com/login

## Phạm vi test
- Chức năng đăng nhập (Login)
- Các kịch bản hợp lệ và không hợp lệ

## Nội dung thực hiện
- Viết test case cho các kịch bản đăng nhập
- Thực hiện test thủ công (manual testing)
- Ghi nhận lỗi và viết bug report

## Các file trong project
- `test-cases.xlsx`: Danh sách test case cho chức năng Login
- `bug-report.xlsx`: Bug report mô tả lỗi phát hiện trong quá trình test
- `index.html`: Trang giới thiệu project (GitHub Pages)

## Công cụ sử dụng
- Excel

## Ghi chú
Project được thực hiện nhằm mục đích học tập và thực hành, không phải sản phẩm thương mại.
